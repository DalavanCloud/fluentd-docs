# Fluentd Enterprise Support Portal

This article describes the Enterprise support program provided by [Treasure Data](https://www.treasuredata.com/fluentd/).

> If you are a Fluentd Service provider wanting to get listed here, please refer to the [ENTERPRISE PROVIDER](https://github.com/fluent/fluentd/blob/master/ENTERPRISE_PROVIDERS.md) guidelines

how to access Fluentd's enterprise support portal.

NOTE: This feature is limited to <a href="https://www.treasuredata.com/fluentd_enterprise">Fluentd Enterprise</a> customers.

## Account Creation

Treasure Data representative will create the access credential for you.

## Support Portal Access

Please access to [console.treasuredata.com/fluentd](https://console.treasuredata.com/fluentd), and type in your login credential.

<center>
<a href="https://console.treasuredata.com/fluentd"><img src="/images/fluentd-enterprise-support.png" width="100%" /></a>
</center>

## Submit a Ticket

After you logged in, please click `Submit a request` link.

<center>
<a href="https://console.treasuredata.com/fluentd"><img src="/images/fluentd-support-portal1.png" width="80%" /></a>
</center>

Then, please input your inquiry's `Subject`, `Description`, `Priority`, and `Attachment`. We **HIGHLY** recommend to attach the log file (`/var/log/td-agent/td-agent.log`) and config file (`/etc/td-agent/td-agent.conf`), since that will be the first question we always ask for you.

<center>
<a href="https://console.treasuredata.com/fluentd"><img src="/images/fluentd-support-portal2.png" width="70%" /></a>
</center>
